# 🚀 Smart Anomaly Detection & Threat Analysis System

## 📌 Overview
This is a major project focused on detecting abnormal patterns in data using Machine Learning and Django.

## 💡 Features
- User Registration with OTP Verification
- CSV Dataset Upload
- Multiple ML Models (SVM, Naive Bayes, AutoEncoder, LSTM)
- Performance Comparison
- Graph Visualization
- Anomaly Prediction

## ⚙️ Tech Stack
- Python, Django
- HTML, CSS, JavaScript
- Scikit-learn, TensorFlow
- Pandas, NumPy, Matplotlib

## 🧠 Concept
Instead of using a single model, this system compares multiple algorithms to identify the best-performing one.
